Privacy by design


The Cobalt browser has been designed from the ground up to provide a privacy-focused experience. The only server it connects to as a small script that says someone new is using Cobalt, and no other data. In fact, the only data it sends is a single "1". This is so we can find out how many people are using Cobalt.


Open Source

All Cobalt code is free and open source for anyone to inspect and review. It is all licensed under MIT and anyone can review all of the code.


Modern


Cobalt is based on Electron, which supports almost all features of Chromium with peace of mind. Cobalt also has an elegant user interface with no unnecessary distractions.
